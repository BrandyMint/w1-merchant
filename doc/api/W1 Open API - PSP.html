Создание мерчанта

POST /OpenApi/merchant

Пример тела запроса:

{

     // Наименование Merchant

     "MerchantTitle":"Merchant",

     // Категория услуг Merchant

     "MerchantCategory":"Cars",

     // Url Merchant 

     "MerchantUrl":"http://merchant.com/",

     // Предпочтительная валюта

     "WithdrawCurrencyId": 643,

     // Страна резидента

     "LocalCountry": "RU"

}

Пример ответа:

{

   // Номер дочернего кошелька

   "PrincipalUserId":100000000000,

   // Дата создания

   "CreateDate":"2012-11-26T06:59:17.863Z",

   // Примечание к кошельку

   "Description":"Principal user",

   // Идентификатор валюты дочернего кошелька, согласно ISO 4217

   "NativeCurrencyId":643,

   // Название кошелька

   "Title":"Principal",

   // Номер текущего кошелька

   "OwnerUserId":100000000001,

   // Логотип

   "MerchantLogo":"logo",

   // URL мерчанта

   "MerchantUrl":"http://merchant.com/"

}

PspMerchants

Получение списка

GET /OpenApi/psp/merchants?pageNumber={pageNumber}

    &itemsPerPage={itemsPerPage}&searchString={searchString}

Обязательные параметры строки запроса:

  • pageNumber — номер страницы, начиная с 1
  • itemsPerPage — количество элементов на странице

Необязательные параметры строки запроса:

  • searchString — строка для поиска по наименованию и url PspMerchant - a.

Пример ответа:

{

   "Items":[ // PspMerchants

      {

         // Идентификатор PspMerchant

         "PspMerchantId":"212239282346",

         // Наименование PspMerchant

         "PspMerchantTitle":"Psp Merchant",

         // Url PspMerchant 

         "PspMerchantUrl":"http://pspmerchant.com/",

         // Категория услуг PspMerchant 

         "PspMerchantCategoryId":"Cars",

         //

         "PspMerchantAccountTypeId":"Personal",

         // Статус модерации PspMerchant 

         "PspMerchantApprovalStateId":"Approved",

         // Статус PspMerchant 

         "PspMerchantIsActive":"1"

      },

      ... // остальные PspMerchants на запрошенной странице

   ],

   // Общее число PspMerchants,

   // удовлетворяющих заданным критериям

   "TotalCount":524

}

Создание PspMerchant

POST /OpenApi/psp/merchants

Пример тела запроса:

{

     // Наименование PspMerchant

     "PspMerchantTitle":"Psp Merchant",

     // Url PspMerchant 

     "PspMerchantUrl":"http://pspmerchant.com/",

     // Категория услуг PspMerchant

     "PspMerchantCategoryId":"Cars",

     //

     "PspMerchantAccountTypeId":"Personal"

}

В случае успеха возвращает HTTP-статус 201 Created и созданного PspMerchant в теле запроса.

Пример ответа:

{

   // Идентификатор PspMerchant

   "PspMerchantId":"212239282346",

   // Наименование PspMerchant

   "PspMerchantTitle":"Psp Merchant",

   // Url PspMerchant 

   "PspMerchantUrl":"http://pspmerchant.com/",

   // Категория услуг PspMerchant 

   "PspMerchantCategoryId":"Cars",

   //

   "PspMerchantAccountTypeId":"Personal",

   // Статус модерации PspMerchant 

   "PspMerchantApprovalStateId":"Approved",

   // Статус PspMerchant 

   "PspMerchantIsActive":"1"

}

Удаление PspMerchant

POST /OpenApi/psp/merchants/{pspMerchantId}

Тело запроса и ответа пустое.
