Получение баланса

GET /OpenApi/balance

GET /OpenApi/balance/{currencyId}

Возвращаются остатки по указанной валюте или по всем валютам, если валюта {currencyId} не указана. Пример ответа (для запроса без указания валюты):

[

  {

     // Идентификатор валюты, согласно ISO 4217:

     "CurrencyId": 643,

     // Остаток:

     "Amount": 6.0000,

     // Окрашенная часть остатка, на которую

     // распространяются определенные ограничения:

     "SafeAmount": 0.0000,

     // Часть остатка, заблокированная на время совершения

     // какой-либо расходной операции:

     "HoldAmount": 0.0000,

     // Доступный размер кредитного лимита:

     "Overdraft": 0.0000,

     // Сумма, доступная для совершения расходных операций:

     "AvailableAmount": 6.0000,

     // Является ли данная валюта основной:

     "IsNative": false

  },

  ... // Список балансов в различных валютах

]

Создать блокировку средств

POST /OpenApi/balance/hold

Создает блокировку средств в указанной валюте на кошельке сессии.

Тело запроса:

{

   // Идентификатор валюты, согласно ISO 4217

   "CurrencyId": 643,

   // Сумма

   "Amount": 6.0000,

   // Описание

   "Description": "Тестовая блокировка средств",

   // Дата, до которой действует блокировка

   "ExpireDate": "2014-09-03T10:00:00",

   // Внешний идентификатор блокировки баланса

   "ClentExternalId": "identifier1"

}

Код ошибки

Описание

EXPIRE_DATE_RANGE_ERROR

Дата автоматического снятия блокировки указана вне требуемого диаппазона.

USER_ID_NOT_FOUND

Кошелек не найден.

CURRENCY_ID_NOT_FOUND

Указанная валюта не найдена.

CURRENCY_IS_NOT_ACTIVE

Указанная валюта отключена.

OPERATION_BANNED

Операция запрещена для выполнения пользователю.

BALANCE_IS_INSUFFICIENT

Недостаточно денег на счете для проведения данной операции.

BALANCEHOLD_EXTERNALID_ALREADY_EXISTS

Блокировка средств с данным идентификатором уже существует.

Получение информации о блокировке средств

GET /OpenApi/balance/hold?externalId={externalId}

Информация о блокировке средств.

ExternalId - внешний идентификатор блокировки.

Тело ответа:

{

   // Внешний идентификатор блокировки баланса

   "ExternalId": "identifier1",

   // Дата создания блокировки баланса

   "CreateDate": "2014-09-03T10:00:00",

   // Дата автоматического снятия блокировки баланса

   "ExpireDate": "2014-11-03T15:00:00",

   // Состояние баланса

   "BalanceHoldStateId": "Created"

}

Код ошибки

Описание

RESOURCE_NOT_FOUND

Блокировка средств не найдена.

Снять блокировку средств

POST /OpenApi/balance/hold/cancel?externalId={externalId}

Отменяет указанную блокировку средств.

ExternalId - внешний идентификатор блокировки.

Тело ответа:

{

   // Внешний идентификатор блокировки баланса

   "ExternalId": "identifier1",

   // Дата создания блокировки баланса

   "CreateDate": "2014-09-03T10:00:00",

   // Дата автоматического снятия блокировки баланса

   "ExpireDate": "2014-11-03T15:00:00",

   // Состояние баланса

   "BalanceHoldStateId": "Created"

}

Код ошибки

Описание

RESOURCE_NOT_FOUND

Блокировка средств не найдена.

BALANCE_HOLD_STATE_ID_CHANGE_ERROR

Смена состояния на требуемое невозможна.
