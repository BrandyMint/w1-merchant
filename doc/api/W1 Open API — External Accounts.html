Получение списка платежных средств

GET /OpenApi/externalAccounts

Возвращает список платежных средств.

Пример тела ответа:

{

   //

   "Items": [

     {

       "ExternalAccountId": 1000001,

       "CreateDate": 2014-01-0100:00:01,

       "Title": "Название",

       "MasterField": "410048203204732934",

       "ExpireDate": 2024-03-14T01:00:13,

       "PaymentTypeId": "CreditCardRUB",

       "LastUsedDate": 2024-03-14T01:00:13,

       "IsDefault": true

     },

     ...

   ]

}

Удаление платежного средства

DELETE /OpenApi/externalAccounts/{externalAccountId}

Тело ответа пусто.

Изменение платежного средства

PUT /OpenApi/externalAccounts/{externalAccountId}

Пример тела запроса:

{

   "Title":"Название платежного средства",

   "Color":"FACE8D", //цвет привязанного инструмента

   "IsDefault":true

}

Тело ответа пусто.
