Для создания шаблона сначала создается платеж на провайдера по которому будет создан шаблон. Затем заполняются его платежные поля. Затем вызывается метод PUT OpenApi/payments/?providerId=SaveTemplate с параметрами {"FieldId":"Title","Value":"название шаблона"} {"FieldId":"PaymentId", "Value":"ID созданного платежа"} - сюда подставляем платеж с заполненными полями.
Поля провайдера SaveTemplate:     
TemplateId - Идентификатор шаблона, передается для изменения платежа
TaskId - Идентификатор задания, используется при задании расписания шаблону. Задание предварительно создается методом https://api.w1.ru/OpenApi/Help/Api/POST-tasks
PaymentId - Идентификатор платежа, на основе которого формируется шаблон
Title - Название шаблона
ResultTemplateId - Это выходной параметр, соответствует идентификатору шаблона
