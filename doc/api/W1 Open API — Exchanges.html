Получение курсов обмена валют

GET /OpenApi/exchanges/rates

Пример тела ответа:

{

   "Items":[

      {

         // Идентификатор списываемой валюты, согласно ISO 4217

         "SrcCurrencyId":643,

         // Идентификатор получаемой валюты, согласно ISO 4217

         "DstCurrencyId":980,

         // Курс обмена валют

         "Rate":3.8234,

         // Процент комиссии за обмен валют

         "CommissionRate":0.0000

      },

      ...

   ]

}

Создание операции обмена валют

POST /OpenApi/exchanges

Пример тела запроса:

{

   // Идентификатор списываемой валюты, согласно ISO 4217

   "SrcCurrencyId":643,

   // Идентификатор получаемой валюты, согласно ISO 4217

   "DstCurrencyId":980,

   // Сумма в получаемой валюте

   "DstAmount":1.23,

   // Необязательно: внешний уникальный идентификатор операции

   "ExternalId":123

}

В случае успеха возвращает HTTP-статус 201 Created и созданную операцию в теле ответа.

Пример тела ответа можно посмотреть в пункте Получение деталей операции.

Получение деталей операции

GET /OpenApi/exchanges/{operationId}

Пример тела ответа:

{

   // Внутренний идентификатор операции в системе

   "OperationId":2458,

   // Дата создания операции

   "CreateDate":"2012-04-13T10:54:18.657Z",

   // Состояние операции

   "OperationStateId":"Accepted",

   // Курс, по которому произведен обмен

   "Rate":3.8234,

   // Идентификатор списанной валюты, согласно ISO 4217

   "SrcCurrencyId":"643",

   // Сумма в списанной валюте

   "SrcAmount":4.70,

   // Идентификатор полученной валюты, согласно ISO 4217

   "DstCurrencyId":"980",

   // Сумма в полученной валюте

   "DstAmount":1.23,

   // Необязательно: внешний уникальный идентификатор операции

   "ExternalId":123

}
