Получение списка уведомлений

GET /OpenApi/notifications/

?pageNumber={pageNumber}&itemsPerPage={itemsPerPage}

&isRead={isRead}&formattingLevel={formattingLevel}

&priority={priority}&searchString={searchString}

Обязательные параметры строки запроса:

  • pageNumber — номер страницы, начиная с 1
  • itemsPerPage — количество элементов на странице

Необязательные параметры строки запроса:

  • isRead —  фильтр по состоянию уведомления
  • true — только прочитанные
  • false — только непрочитанные
  • formattingLevel — уровень форматирования уведомления
  • PlainText — возвращается без HTML-тегов
  • NoImages — возвращается без изображений
  • Full — возвращается полностью
  • priority — фильтр по приоритету уведомления
  • High
  • Normal
  • Low
  • searchString — фильтр по частичному совпадению с текстом уведомления

Пример тела ответа:

{

   "TotalCount": 123,

   "Items":[

      {

         "NotificationId": 12345,

         "CreateDate": "2012-01-10T06:10:14.52Z",

         "NotificationPriorityId": "Normal",

         "NotificationTypeId": "System",

         "NotificationSubtypeId": "System",

         "Subject": "Заголовок уведомления",

         "Body": "Текст уведомления",

         "IsRead": false

      },

      ...

   ]

}

Получение уведомления по идентификатору

GET /OpenApi/notifications/{notificationId}

/?formattingLevel={formattingLevel}

Необязательные параметры строки запроса:

  • formattingLevel — уровень форматирования уведомления
  • PlainText — возвращается без HTML-тегов
  • NoImages — возвращается без изображений
  • Full — возвращается полностью

Пример тела ответа:

{

   "NotificationId": 12345,

   "CreateDate": "2012-01-10T06:10:14.52Z",

   "NotificationPriorityId": "Normal",

   "NotificationTypeId": "System",

   "NotificationSubtypeId": "System",

   "Subject": "Заголовок уведомления",

   "Body": "Текст уведомления",

   "IsRead": false

}

Изменение состояния уведомления

POST /OpenApi/notifications/state

Изменяет состояние уведомлений с указанными идентификаторами.

Возможные состояния уведомления:

  • Read - прочитано;
  • Unread - непрочитано.

Пример тела запроса:

{

   // Идентификаторы уведомлений

   "Items": [ 123, 456, 789 ],

   "NotificationStateId": "Read"

}

Тело ответа пустое.

Удаление уведомления

DELETE /OpenApi/notifications

Удаляет состояние уведомлений с указанными идентификаторами.

Пример тела запроса:

{

   // Идентификаторы уведомлений

   "Items": [ 123, 456, 789 ]

}

Тело ответа пустое.

Отправка уведомления

POST /OpenApi/notifications

Отправляет уведомление указанного типа с указанными значениями шаблона уведомления

Пример тела запроса:

{

   // Тип уведомления

   "EventTypeId": "[Тип уведомления]",

   // Параметры шаблона уведомления

   "EventParams":[

      {

         "PlaceHolderId": "[Имя параметра]",

         "PlaceHolderValue": "[Значение параметра]"

      },

      ...

   ]

}

Тело ответа пустое.
